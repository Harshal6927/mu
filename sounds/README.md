# Sounds Directory

Place your audio files here (WAV, MP3, OGG, FLAC, M4A).

The soundboard will automatically detect and load all audio files from this directory.

## Organization Tips

You can organize files in subdirectories:
```
sounds/
├── music/
│   ├── song1.mp3
│   └── song2.mp3
├── effects/
│   ├── airhorn.wav
│   └── explosion.wav
└── memes/
    └── rickroll.mp3
```

## File Naming

- Use descriptive filenames (e.g., `epic_entrance.mp3`)
- Avoid special characters
- The filename (without extension) will be the sound name in the app

## Recommended Audio Format

- **Format**: WAV or MP3
- **Sample Rate**: 44100 Hz or 48000 Hz
- **Bit Depth**: 16-bit or higher
- Keep file sizes reasonable for quick loading
